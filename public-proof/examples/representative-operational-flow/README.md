# Representative Operational Flow

This package is a synthetic, public-safe example of the Fixera Core operational flow.

It exists to show how a job moves through the public model without exposing production data.

## Files

- `synthetic-job.json` - one synthetic job record with representative fields
- `lead-event-example.json` - one synthetic first-party lead event with representative fields
- `validation.md` - the checks used to keep the sample public-safe
- `provenance.md` - what the sample is based on and how it was transformed
