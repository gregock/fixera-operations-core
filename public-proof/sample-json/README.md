# Sample JSON

This folder documents the synthetic JSON publication rule for the Portfolio Edition.

The current synthetic JSON samples live in:

- [public-proof/examples/representative-operational-flow/synthetic-job.json](../examples/representative-operational-flow/synthetic-job.json)
- [public-proof/examples/representative-operational-flow/lead-event-example.json](../examples/representative-operational-flow/lead-event-example.json)

They are kept with the representative flow so the example, validation notes, and provenance stay together.

Only include records that are invented for publication and verified to be safe.

Do not add:

- production exports
- client identifiers
- operational logs
- timestamps that can be tied to a live event
